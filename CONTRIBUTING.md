# Contributing

This repository is a Proof of Concept web miner for ZKGRM. It may contain bugs or security-sensitive UX issues. Contributions are welcome for the benefit of the whole community.

## Useful Contributions

- Safer TonConnect flows.
- Better browser compatibility.
- UI/UX improvements.
- Mining performance improvements.
- Documentation in English or Russian.

## Pull Request Checklist

- Explain the behavior change.
- Test in at least one modern desktop browser.
- Do not commit mnemonics, wallet keys, or private deployment URLs.
- Keep legacy seed mode clearly marked as unsafe for everyday wallets.
